## 🚀 Projetos

### 🏆 Destaque

<div align="center">

[![Sistema Empresarial](https://github-readme-stats.vercel.app/api/pin/?username=cristianlabs&repo=sistema-empresarial&theme=tokyonight&hide_border=true)](https://github.com/cristianlabs/sistema-empresarial)

</div>

**Sistema Empresarial** é uma plataforma SaaS modular construída com tecnologias modernas:

| Módulo | Descrição |
|---|---|
| 🎯 **CRM** | Pipeline de deals, contatos, atividades e negócios |
| 📈 **Vendas** | Cotação → Proposta → Pedido com análise de liberação |
| 📦 **Estoque** | Controle de produtos, movimentações e alertas |
| 💰 **Financeiro** | Contas a pagar/receber, fluxo de caixa, comissões |
| 🧾 **NF-e** | Notas fiscais, regras CFOP, controle de canhoto |
| 💬 **Comunicação** | WhatsApp Business via Evolution API + Chatbot |
| 👥 **RH** | Férias, requisições, documentos de colaboradores |
| 🎓 **Treinamentos** | LMS interno com controle de progresso |

### 💰 Recente — WalletCore

<div align="center">

[![WalletCore](https://github.com/cristianlabs/WalletCore/blob/main/README.md)

</div>

**WalletCore** é um app de controle financeiro pessoal com dashboard intuitivo e metas inteligentes:

| Funcionalidade | Descrição |
|---|---|
| 📊 **Dashboard** | Visão geral de receitas, despesas e saldo em tempo real |
| 🎯 **Metas** | Defina e acompanhe metas de economia por categoria |
| 🏷️ **Categorias** | Organize transações com categorias e subcategorias customizáveis |
| 📈 **Relatórios** | Gráficos mensais, comparativos e projeções futuras |
| 💳 **Contas** | Multi-conta bancária com fluxo de caixa unificado |
| 🔔 **Alertas** | Notificações de gastos acima do orçamento definido |

> **Stack:** Next.js · TypeScript · Prisma · PostgreSQL · Supabase · Tailwind CSS · NextAuth

### ⏳ Em breve — TaskFlow

<div align="center">

![Em desenvolvimento](https://img.shields.io/badge/Status-Em_desenvolvimento-F59E0B?style=for-the-badge&logo=github&logoColor=white)

</div>

**TaskFlow** será um gerenciador de tarefas moderno com foco em produtividade pessoal e colaboração em equipe.

| Funcionalidade planejada | Descrição |
|---|---|
| 📋 **Boards** | Kanban drag-and-drop com colunas customizáveis |
| 🏷️ **Tags & Prioridades** | Organize por tags, urgência e data de entrega |
| 👥 **Colaboração** | Atribua tarefas, comente e acompanhe em equipe |
| ⏱️ **Time tracking** | Registre o tempo gasto por tarefa e projeto |
| 📊 **Produtividade** | Métricas semanais, foco e histórico de conclusões |
| 🔔 **Lembretes** | Notificações inteligentes por e-mail e push |

> **Stack prevista:** Next.js · TypeScript · Prisma · PostgreSQL · Supabase · Tailwind CSS · Zustand
