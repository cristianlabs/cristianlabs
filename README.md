## 🧑‍💻 Sobre mim

Sou desenvolvedor full stack com foco em **produtos SaaS** e **sistemas empresariais**. Construo desde a arquitetura do banco de dados até a interface do usuário, priorizando código tipado, escalável e de fácil manutenção.

Atualmente desenvolvendo um **ERP/CRM modular** completo — com pipeline de vendas, controle de RH, financeiro, estoque, NF-e e integração com WhatsApp Business.

- 🔭 Trabalhando em: **Sistema Empresarial** — plataforma SaaS multi-módulo com Next.js + Supabase
- 🌱 Explorando: **IA generativa aplicada a processos de negócio**
- 💡 Filosofia: *Menos configuração, mais produto. Código limpo, deploy rápido.*

## 🚀 Projeto em destaque

<div align="center">

[![Sistema Empresarial](https://github-readme-stats.vercel.app/api/pin/?username=SEU_USERNAME&repo=sistema-empresarial&theme=tokyonight&hide_border=true)](https://github.com/SEU_USERNAME/sistema-empresarial)

</div>

**Sistema Empresarial** é uma plataforma SaaS modular construída com tecnologias modernas:

| Módulo | Descrição |
|---|---|
| 🎯 **CRM** | Pipeline de deals, contatos, atividades e negócios |
| 📈 **Vendas** | Cotação → Proposta → Pedido com análise de liberação |
| 📦 **Estoque** | Controle de produtos, movimentações e alertas |
| 💰 **Financeiro** | Contas a pagar/receber, fluxo de caixa, comissões |
| 🧾 **NF-e** | Notas fiscais, regras CFOP, controle de canhoto |
| 💬 **Comunicação** | WhatsApp Business via Evolution API + Chatbot |
| 👥 **RH** | Férias, requisições, documentos de colaboradores |
| 🎓 **Treinamentos** | LMS interno com controle de progresso |
